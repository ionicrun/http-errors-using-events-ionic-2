## Getting started

```bash
# download or clone
$ git clone https://github.com/ionicrun/in-memory-web-api-ionic-2-base.git
$ cd in-memory-web-api-ionic-2-base

# install the packages
$ npm i

# serve or run
$ ionic serve

# or
# ionic cordova run ios/android

```
